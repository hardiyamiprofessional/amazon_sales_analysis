# Amazon Sales Data Analysis | Python Project

This project analyzes Amazon sales data to uncover business insights and product performance patterns.

## Key Highlights

* T-shirt category generated the highest total sales, followed by Shirt and Blazer
* Casual fashion products dominate overall revenue
* High number of cancelled orders in T-shirt and Shirt categories

## Key Insight

Although T-shirt leads in total sales, it also has the highest cancellation rate.
This may indicate issues in product quality, inventory management, or customer expectations.

## Additional Findings

* Certain cities contribute significantly to overall revenue
* Sales trends vary across months, suggesting seasonal patterns

## Dataset

* Contains over 128,000 transaction records
* Cancellation rate is approximately 14%
